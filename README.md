<h2>UniPath - Gruppo 12</h2>

Applicazione mobile sviluppata per il corso di Mobile Programming, finalizzata all'organizzazione dello studio universitario e al monitoraggio accademico. L'app funge da assistente personale per gestire corsi, scadenze, sessioni di studio e obiettivi di apprendimento.

<h3>Caratteristiche Principali</h3>

- Gestione Corsi: Monitoraggio di esami, CFU, docenti e stato di avanzamento per ogni insegnamento.
- Tracking Esami & Scadenze: Visualizzazione di appelli e date importanti con sistema di priorità.  
- Pianificazione Intelligente: Organizzazione di sessioni di studio giornaliere o settimanali.  
- Obiettivi e Statistiche: Visualizzazione del progresso tramite riepiloghi grafici e statistiche sul tempo dedicato allo studio.  
- Framework: React Native / Flutter (da specificare).  ----> Modificare 
- Framework: Flutter  
- Stato & Navigazione: Gestione avanzata dei flussi e passaggio dati tra schermate.  
- Persistenza: Gestione dei dati locale per un utilizzo offline.
